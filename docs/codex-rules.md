## Codex Rules

1. Never ask Codex to build the whole app at once

2. Always use small prompts

3. One feature per prompt

4. One file group per prompt

5. Ask for review before next step

6. Never allow creative assumptions

7. Always specify:

* exact files
* exact routes
* exact fields
* exact output

8. Prefer edits over rewrites

9. Verify after every step

10. Production-ready only
